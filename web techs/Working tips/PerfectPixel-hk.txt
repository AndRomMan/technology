Список горячих клавиш:
Alt + S - Показать/Скрыть слой
Alt + C - Заблокировать/Разблокировать слой
Alt + H - Показать/Скрыть панель управления целиком
= - Увеличить непрозрачность
- - Уменьшить непрозрачность
↑ - Переместить слой вверх
↓ - Переместить слой вниз
← - Переместить слой вправо
→ - Переместить слой влево
Shift + ↑, ↓, ←, → - Перемесить слой с 10-кратной скоростью
